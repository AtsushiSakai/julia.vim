# julia.vim

A vim plugin for Julia.

functions:

- Asynchronous lint checker. 

# Requirements

- [Julia](https://julialang.org/)

- [Lint.jl](https://github.com/tonyhffong/Lint.jl)

# Install

- NeoBundle

> NeoBundle 'AtsushiSakai/julia.vim'
